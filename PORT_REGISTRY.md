# Bearing Port Registry

All local development ports used by Bearing services.

| Port | Service | Command |
|------|---------|---------|
| 8080 | Bearing API (FastAPI) | `make run` |
| 4201 | Bearing Prototype UI (Vite) | `make proto-dev` |
| 4210 | Customer Documentation (VitePress) | `make docs-customer-dev` |
| 4211 | Internal Documentation (VitePress) | `make docs-internal-dev` |
