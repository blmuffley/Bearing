## Dashboard Screenshots

### Pre-Remediation Dashboard

The initial assessment dashboard showing current state health scores, findings, and technical debt estimate. Use this view when presenting the "current state" to a customer.

![Dashboard Pre-Remediation](/screenshots/01_dashboard_pre.png)

### Post-Remediation Dashboard

The dashboard after remediation, showing improved scores. Use this view to illustrate the value of engagement.

![Dashboard Post-Remediation](/screenshots/07_dashboard_post.png)

### Findings View

The detailed findings table used during deep-dive sessions to walk through specific issues with the customer's technical team.

![Findings](/screenshots/02_findings.png)

### Fusion Findings

When Pathfinder data is available, the fusion findings view provides high-impact findings only detectable through combined static and behavioral analysis.

![Fusion Findings](/screenshots/08_fusion_findings.png)
