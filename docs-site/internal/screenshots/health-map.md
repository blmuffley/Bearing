# Health Map Screenshots

Screenshots of the Bearing prototype health map / radar visualization.

## Dimension Radar Chart

The radar chart showing all eight CMDB health dimensions with scores plotted on a 0-100 scale. This provides an at-a-glance view of where the instance is strong and where it needs work.

![Health Map](/screenshots/03_health_map.png)

## Key Elements

- **Eight dimensions** -- Completeness, Accuracy, Currency, Classification, Relationships, CSDM Compliance, Duplicates, Orphans
- **Color coding** -- Green (75+), Amber (40-74), Red (below 40)
- **Weighted scores** -- Each dimension has a configurable weight that contributes to the overall score
